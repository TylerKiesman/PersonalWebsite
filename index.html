Number Guessing Game Although I wouldn't consider this a major project by any means, this small exercise introduced me to JavaScript and jQuery. It's a basic guess the number game that I created to get a better feel for how JavaScript interacts with a webpage and to get used to the syntax of jQuery. When a user guesses at the random number the webpage responds by giving hints to the user until they guess the correct number. A link to this game can be found here.
selfLook selfLook is a social media platform that I'm still in the works of finishing up. It is set to have a feeling similar to Instagram or Facebook and would involve users being able to create accounts and follow others on the platform. They’ll be able to view the posts of people that they follow and can comment on them. It’s currently being built using Python and the Flask framework and utilizes the SQLAlchemy toolkit to create and access an SQL database.
